# Hướng Dẫn Nghiệm Thu (Walkthrough): AI Content Factory (Phase 3 - Local Web UI Console)

Hệ thống đã hoàn thành việc tích hợp toàn bộ luồng quy trình vào một **Giao diện Web (Web UI) cục bộ cao cấp**, giúp người dùng không có kiến thức kỹ thuật có thể dễ dàng sản xuất video chỉ với 1 cú click chuột.

---

## 1. Kết Quả Công Việc Đã Thực Hiện

Chúng ta đã tạo lập thêm các cấu trúc phục vụ giao diện Web:
1.  **src/ui_server.py:** Máy chủ FastAPI quản lý các API endpoints (sinh kịch bản, chạy render, đồng bộ DB) và phân phối tệp giao diện tĩnh.
2.  **src/web/index.html:** Giao diện người dùng Web (gồm các ô cấu hình API key, IP máy GPU, ô nhập chủ đề kịch bản, bảng log động, trình phát video HTML5 và bảng báo cáo PostgreSQL terminal).
3.  **docs/non_tech_user_guide.md:** Hướng dẫn sử dụng chi tiết dành cho người không chuyên.

---

## 🌐 2. Cách Khởi Chạy Giao Diện Web UI & Kiểm Chứng

Để khởi động giao diện và kiểm thử:

1.  **Chạy database PostgreSQL trong Docker Desktop:**
    ```powershell
    docker run --name content-creator-db -e POSTGRES_PASSWORD=postgres -p 5432:5432 -d pgvector/pgvector:pg16
    ```
2.  **Khởi động máy chủ Web UI:**
    ```powershell
    $env:PYTHONUTF8=1
    .venv\Scripts\python.exe src/ui_server.py
    ```
    *Nhật ký khởi chạy thành công:*
    ```text
    Launching AI Content Factory local server on http://localhost:8000
    INFO:     Started server process [1234]
    INFO:     Waiting for application startup.
    INFO:     Application startup complete.
    INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
    ```

3.  **Truy cập và sử dụng:**
    *   Mở trình duyệt web của bạn và truy cập **[http://localhost:8000](http://localhost:8000)**.
    *   Dán mã `OPENAI_API_KEY` của bạn và nhập chủ đề video (ví dụ: "Tam giác quỷ Bermuda").
    *   Nhấn **Bắt đầu sản xuất video** (Generate Video).

## 🏁 3. Xác Minh Kết Quả Đầu Ra Trực Quan trên Giao diện Web:

Khi quá trình sinh video hoàn tất, bạn sẽ kiểm chứng được:

1.  **Phát video trực tiếp:** Trình phát video trên giao diện web sẽ hiển thị video [output.mp4](file:///c:/Users/uytpv/works/content-creator/src/web/static/output.mp4) vừa được sinh ra. Bạn có thể nhấn phát ngay để kiểm tra hình ảnh zoompan, phụ đề đồng bộ chạy chữ và giọng đọc AI.
2.  **Terminal Báo Cáo SQL:** Khung báo cáo cơ sở dữ liệu ở chân trang sẽ cập nhật thông tin chi tiết các bản ghi đã ghi nhận vào PostgreSQL (bảng `scripts`, `entities`, `assets`, `videos`).
3.  **Thư mục lưu trữ assets:** Toàn bộ ảnh PNG và audio giọng nói MP3 sinh ra được lưu trữ lâu dài tại thư mục [src/mock_assets/](file:///c:/Users/uytpv/works/content-creator/src/mock_assets/) để phục vụ tái sử dụng sau này.
