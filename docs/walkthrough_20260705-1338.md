# Hướng Dẫn Nghiệm Thu (Walkthrough): AI Content Factory (Phase 2 - Local GPU Image Gen)

Hệ thống đã được tích hợp khâu **Sinh ảnh tự động từ xa/cục bộ (Local GPU Image Gen)** nhằm tự động hóa hoàn toàn quy trình từ văn bản đầu vào đến video xuất ra, đồng thời hỗ trợ cơ chế dự phòng thông minh.

---

## 1. Kết Quả Công Việc Đã Thực Hiện (Giai đoạn 2)

Chúng ta đã bổ sung và nâng cấp các cấu trúc sau:

1.  **src/render/image_gen.py:** **[MỚI]** Module kết nối với API của Stable Diffusion WebUI thông qua địa chỉ cấu hình `SD_API_URL`. Nếu máy chủ GPU offline, hệ thống tự động băm (hash) prompt để vẽ ra ảnh nháp (placeholder) có màu sắc nhất quán làm dự phòng.
2.  **src/render/video_composer.py:** **[NÂNG CẤP]** Tự động phát hiện nếu phân cảnh thiếu hình ảnh (`visual_asset_path`). Hệ thống sẽ tự động gọi `image_gen` để sinh ảnh từ `visual_prompt` (hoặc lấy kịch bản chữ làm prompt dự phòng) và áp dụng hiệu ứng chuyển động tương ứng.
3.  **src/timeline_example.json:** **[NÂNG CẤP]** Chuyển đổi kịch bản sang dạng động hoàn toàn: Gỡ bỏ toàn bộ đường dẫn ảnh tĩnh và thay bằng `"visual_prompt"` cho mỗi phân cảnh để AI tự động vẽ.

---

## 2. Cách Thiết Lập Và Chạy Thực Tế

### Bước 1: Cấu hình địa chỉ GPU (Nếu chạy máy GPU riêng trong mạng LAN)
Khi bạn chạy script trên máy dựng video, bạn có thể thiết lập biến môi trường `SD_API_URL` trỏ tới IP của máy chạy GPU (mặc định là `http://localhost:7860` nếu chạy chung máy).
*   **PowerShell:**
    ```powershell
    $env:SD_API_URL="http://192.168.1.XX:7860" # Thay XX bằng IP máy GPU của bạn
    ```

*Lưu ý trên máy GPU:* Cần chạy Stable Diffusion WebUI với cờ `--api` (ví dụ: `webui-user.bat` sửa dòng `set COMMANDLINE_ARGS=--api --listen`).

### Bước 2: Chạy kiểm thử toàn bộ hệ thống
Để chạy thử nghiệm toàn bộ luồng dựng video:
```powershell
$env:PYTHONUTF8=1
.venv\Scripts\python.exe src/main.py
```

*   **Nếu máy GPU đang bật:** Hệ thống sẽ gọi máy GPU vẽ ảnh thật độ phân giải Full HD (1920x1080) và tự động ghép vào video.
*   **Nếu máy GPU đang tắt:** Hệ thống sẽ ghi nhận cảnh báo kết nối thất bại, tự động sinh ảnh nháp đại diện cho prompt đó và hoàn tất render file video `output.mp4` để bạn kiểm thử luồng hoạt động mà không bị dừng chương trình.
