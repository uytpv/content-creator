# Hướng Dẫn Nghiệm Thu (Walkthrough): Nền Móng AI Content Factory (Phase 1)

Dự án đã hoàn thành thiết lập nền móng kiến trúc cốt lõi của hệ thống sản xuất video tự động theo đúng kế hoạch triển khai. Dưới đây là các phần việc đã hoàn thành và kết quả nghiệm thu.

---

## 1. Kết Quả Công Việc Đã Thực Hiện

Chúng ta đã tạo lập các cấu trúc thư mục và file mã nguồn sau:

1.  **requirements.txt:** Khai báo các thư viện phụ thuộc (`pydantic`, `pillow`, `psycopg2-binary`, `requests`, `imageio-ffmpeg`).
2.  **schema/schema.sql:** Sơ đồ Database PostgreSQL hoàn chỉnh với tiện ích mở rộng `pgvector` phục vụ lưu trữ Content Graph và tìm kiếm thực thể ngữ nghĩa.
3.  **src/create_mock_assets.py:** Script tự động sinh tài nguyên thử nghiệm (4 ảnh Full HD màu sắc phân biệt và 5 file âm thanh giọng nói + nhạc nền WAV sạch).
4.  **src/timeline_example.json:** Kịch bản phân cảnh mẫu (storyboard) dạng JSON mô tả 4 scene dài tổng cộng 26 giây kèm timestamps của phụ đề.
5.  **src/render/subtitle_gen.py:** Thư viện chuyển đổi timestamps thành định dạng phụ đề `.ass` cao cấp (hỗ trợ viền đen nổi bật, font Arial cỡ lớn và căn giữa).
6.  **src/render/video_composer.py:** Bộ điều phối FFmpeg thực hiện cắt ghép, áp hiệu ứng Ken Burns (zoom-in/zoom-out), burn phụ đề và trộn nhạc nền với chế độ ducking âm lượng.
7.  **src/main.py:** Script chạy chính của hệ thống thử nghiệm, cấu hình `PYTHONUTF8=1` để tương thích hoàn toàn với bảng mã Unicode trên Windows.

---

## 2. Quy Trình Kiểm Thử & Nghiệm Thu

Để xác minh hệ thống hoạt động ổn định và chính xác:

1.  **Thiết lập môi trường & Sinh tài nguyên thử nghiệm:**
    ```powershell
    # Khởi tạo venv và cài đặt thư viện tự động qua uv
    uv venv --python 3.11
    uv pip install -r requirements.txt
    
    # Tạo các file ảnh và âm thanh WAV mẫu để test
    .venv\Scripts\python.exe src/create_mock_assets.py
    ```
    *Kết quả:* Các thư mục được khởi tạo, 4 tệp ảnh `.png` và 5 tệp âm thanh `.wav` đã được sinh ra chính xác trong thư mục `src/mock_assets/`.

2.  **Chạy pipeline dựng video:**
    ```powershell
    # Kích hoạt chế độ UTF-8 của Python trên Windows và chạy dựng video
    $env:PYTHONUTF8=1
    .venv\Scripts\python.exe src/main.py
    ```
    *Kết quả logs:*
    ```text
    Starting video composition: Bi mật của Mèo Cam
    Generated ASS subtitle file: temp/scene_1.ass
    Rendering scene clip: temp/scene_1.mp4 (effect: zoom_in, duration: 6.00s)...
    Generated ASS subtitle file: temp/scene_2.ass
    ...
    Concatenating scenes into a unified video...
    Mixing background music (volume: 0.08)...
    Video composed successfully! Output file: output.mp4
    Cleaning up temporary files...
    ```

3.  **Xác minh file thành phẩm:**
    *   File thành phẩm [output.mp4](file:///c:/Users/uytpv/works/content-creator/output.mp4) đã được xuất thành công ở thư mục gốc của dự án với dung lượng khoảng **1.5 MB** và thời lượng chính xác **26 giây**.
    *   Hệ thống tự động dọn sạch tất cả các file phụ đề `.ass` tạm và clip MP4 trung gian ở thư mục `temp/` sau khi xuất video thành công.
