# Hướng Dẫn Nghiệm Thu (Walkthrough): AI Content Factory (Phase 3 - End-to-End Integration)

Chúng ta đã hoàn thành tích hợp toàn bộ luồng quy trình: **Từ khóa chủ đề (User Input) -> OpenAI GPT-4o-mini Biên kịch kịch bản phân cảnh -> Lưu dữ liệu PostgreSQL -> Sinh giọng đọc thực (Edge-TTS) -> Sinh ảnh nháp dự phòng (Offline fallback) -> Gọi FFmpeg render ghép nhạc nền -> Lưu vết Assets thành phẩm vào Database -> In báo cáo SQL xác minh.**

---

## 1. Kết Quả Công Việc Đã Thực Hiện

Mã nguồn mới của hệ thống bao gồm:
1.  **src/agents/script_writer.py:** Agent gọi OpenAI API bằng cơ chế `beta.chat.completions.parse` và Pydantic schema để sinh kịch bản JSON timeline chuẩn xác.
2.  **src/db/manager.py:** Quản lý kết nối PostgreSQL bằng `psycopg2`. Tự động khởi tạo cấu trúc bảng từ `schema/schema.sql` và hỗ trợ các hàm ghi/đọc dữ liệu kịch bản, thực thể (Entities), tài nguyên (Assets) và video.
3.  **src/render/video_composer.py:** Dựng video đa phân cảnh, tự động sinh âm thanh nói (Edge-TTS), tự động sinh ảnh (Stable Diffusion API / Fallback), và sao lưu lâu dài các file này vào thư mục `src/mock_assets/` để ghi nhận đường dẫn vào Database.
4.  **src/main.py:** Module chạy chính điều hợp toàn bộ 6 bước hoạt động và in báo cáo xác minh SQL.

---

## 🔍 Hướng Dẫn Cách Kiểm Trực Quan Kết Quả Đầu Ra

### Bước 1: Khởi động Database PostgreSQL ( pgvector )
Chạy lệnh sau trong PowerShell để bật nhanh một database PostgreSQL hỗ trợ vector thông qua Docker:
```powershell
docker run --name content-creator-db -e POSTGRES_PASSWORD=postgres -p 5432:5432 -d pgvector/pgvector:pg16
```
*(Nếu cổng 5432 của bạn đã được sử dụng, đổi thành `-p 5433:5432`, sau đó set thêm biến môi trường `$env:DATABASE_URL="postgresql://postgres:postgres@localhost:5433/postgres"` trước khi chạy)*

### Bước 2: Chạy toàn bộ hệ thống tích hợp
Nạp API key OpenAI của bạn, bật tính năng bảng mã Unicode UTF-8 và chạy chương trình chính:
```powershell
$env:OPENAI_API_KEY="sk-proj-..." # Thay bằng khóa OpenAI thật của bạn
$env:PYTHONUTF8=1
.venv\Scripts\python.exe src/main.py --topic "Bí ẩn kỳ lạ về Tam giác quỷ Bermuda"
```
